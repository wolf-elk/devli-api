## 커밋 컨벤션

| 종류     | 설명                          |
| -------- | ----------------------------- |
| feat     | 새로운 기능 추가              |
| fix      | 버그 수정                     |
| !bc      | 커다란 변경의 경우            |
| !hotfix  | 급한 치명적 오류              |
| refactor | 코드 수정                     |
| comment  | 주석 관련                     |
| docs     | 문서 관련                     |
| test     | 테스트 코드 관련              |
| chore    | 프로덕션 코드와 관련없는 작업 |
